Ingredient Categories

Fourteen hues map every ingredient to a category — produce greens, meat reds, a dairy yellow, frozen blues. Each is applied as a tonal tag (color@12% fill, color@85% text). The hues are tuned for light surfaces; in dark mode the code currently reuses them unchanged, and several wash out — so a luminance-lifted dark set is proposed below.
