Proposed Semantic Tokens

Every token on this card is a proposal — none of these exist in design_tokens.dart yet. They live only as --proposed-* CSS vars so the system can preview them. Each formalizes a gap the app currently fills ad-hoc: a real disabled surface, a focus ring, distinct success/info/warning that don't borrow from freshness, and a scrim for photo overlays.
